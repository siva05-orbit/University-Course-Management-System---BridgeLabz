# 🎓 University Course Management System

## 📌 Overview

This project is a Python-based system that simulates a university course environment. It manages course details, student enrollments, schedule conflicts, and grading analytics.

The system is built using **Object-Oriented Programming (OOP)** concepts and is divided into three modules developed collaboratively.

---

## 👥 Team Structure

### 🔹 Member 1 – Course & Student Module

* Created `Course` and `GradedStudent` classes
* Handles course details and student enrollment
* Calculates total enrolled credits

---

### 🔹 Member 2 – Schedule Conflict Detector

* Detects timetable clashes using course timeslots
* Generates:

  * `timeslot_map`
  * `conflict_report`
  * `conflict_free` students

---

### 🔹 Member 3 – Grading & Analytics

* Parses student marks into structured data
* Calculates GPA using course credits
* Generates class performance report:

  * Average marks per course
  * Top scorer per course

---

## ⚙️ Features

* Course creation and student enrollment
* Conflict detection in schedules
* GPA calculation using weighted credits
* Class performance analytics
* Modular and reusable code structure

---

## 🧠 Concepts Used

* OOP (Classes, Objects, Encapsulation)
* Dictionaries & Nested Dictionaries
* Lists, Sets, Tuples
* Sorting & Lambda functions
* Algorithms (aggregation, conflict detection)

---

## 🚀 How to Run

1. Clone the repository
2. Run the Python file:

   ```bash
   python main.py
   ```
3. Outputs for all 3 modules will be displayed in the console

---

## 📊 Sample Output

* Course details and student credits
* Schedule conflict report
* GPA of each student
* Class report (average + topper per course)
